# Source Code

`match2_demo.py` is a privacy-safe, runnable demonstration of the Match2 workflow.

It uses only the fictional Excel files included in the public repository and demonstrates:

- Job discipline classification through the OpenAI API
- Candidate mapping to the controlled discipline repository
- Discipline-gated candidate shortlisting
- Strict candidate-to-job scoring with reasons and missing skills
- Geographic distance calculation using coordinates and the Haversine formula
- Ranked Excel output generation

The full production program processes private recruitment records and is therefore not published with real data or private operational files.
